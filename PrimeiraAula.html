Explique o HTML.

HTML é uma linguagem de marcação de hipertexto que está presente em todos os dites existentes na web. Pode -se afirmar que não existe desenvolvimento web sem HTML.

O que são as TAGs, descreva exemplos.

As TAgs são comentário HTML, que definem regiões em modelos ou documentos. EX: Heading tags- são TAgs para destacar o conteúdo. Title- são consideradas pelo goolgle como principal conteúdo a ser indexado quando um usuário realiza uma pesquisa.

Explique o CSS.

É um recurso que permite controlar a aparência dos elementos HTML de uma página.

Explique UTF-8 e para que usamos.

É a codificação de caracteres mais comum da World Wide Web. Usamos para garantir que todos os caracteres sejam codificados corretamente.
